1.
  1. Add -
    a. new table
    CREATE TABLE table-name (
      column-name-1 datatype,
      column-name-2 datatype
    );

    b. multiple rows
    INSERT INTO table-name (column-name-1, column-name-2, column-name-3)
    VALUES
      (new-row-1-column-1-data, column-2-data, column-3-data),
      (new-row-2-column-1-data, column-2-data, column-3-data);

    c. one row, data in order
    INSERT INTO table-name VALUES (column-1-data, column-2-data, column-3-data);

  2. Update -
    a. update data
    UPDATE table-name SET column-name=new-value, another-column-name=new-value WHERE column-name=current-value AND another-column-name=current-value;
    b. update table
    ALTER TABLE table-name ADD COLUMN column-name data-type SET DEFAULT default-value;

  3. Delete -
    a. delete data
    DELETE FROM table-name WHERE column-name=value AND column-name=value;
    b. delete column
    ALTER TABLE table-name DROP COLUMN column-name;
    c. delete table
    DROP TABLE table-name;

3. Datatypes -
  - varchar
    - string of random length, name, email, username
  - timestamp
    - date-time data is stored, publish date-time of a blog post
  - int
    - userid, inventory count of particular item
  - boolean
    - true/false, if a user has a paid membership, if a product is in stock
